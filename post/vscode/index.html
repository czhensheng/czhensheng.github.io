快捷键操作

按键 功能
Ctrl+Tab 按住Ctrl+Tab，点击Tab可以循环切换在编辑区里打开的文件

添加文件头模板

对于新建的文件，根据自己设定的文件头版本，可以快速生成文件模板
操作：
文件 -> 首选项 -> 用户代码片段 -> （选择相应的模板） ->
在对应的配置文件中，添加相应的文件头格式即可

个人用例
写完之后，在文件上输入 header 再回车就可直接生成相应的模板
python文件

{
    "HEADER":{
        "prefix": "header",
        "body": [
            "# Author : czhensheng",
            "# Encoding : utf-8",
            "# CreatedTime : $CURRENT_YEAR/$CURRENT_MONTH/$CURRENT_DATE         $CURRENT_HOUR:$CURRENT_MINUTE:$CURRENT_SECOND",
            "# Function :",
            "#           ",
            "$0"
        ],
    }
}

对应生成的模板

# Author : czhensheng
# Encoding : utf-8
# CreatedTime : 2020/02/24 09:43:40
# Function :

配置代码片段快捷键

按F1 或者 Ctrl+Shift+P ，输入 snippets

选择需要新建代码片段的文件类型

格式如下

{
    "代码片段名称": {
        "prefix": "快捷代码片段",
        "body": [
            // 实际插入的代码内容
            // $0 光标停留位置
        ],
        "description": "提示信息描述"
    },
    // "printf":{
    //     "prefix": "plf",
    //     "body": "fmt.Printf(\"$0\")",
    //     "description": "printf"
    // }
}
